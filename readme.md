# Как запустить локально?

1. Скачайте репозиторий
2. Перейдите в него и вызовите:
```bash
npm i
```
3. После успешного скачивания всех зависимостей, выполните сборку:
```bash
npm run build
```
4. Для запуска на localhost выполните команду:
```bash
npm run site
```
